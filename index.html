what the hex is that?

#000000

what's going on here?

pick the colour that matches the hex code

after the #, a hex code shows how much RED (first two values), GREEN (second two values)
and BLUE (final two values) that a colour contains

#ff0000
#00ff00
#0000ff

the possible values are 0 to f - the closer to f, the more of that colour

the three values combine to form other less horrifying colours

#15b1d9
